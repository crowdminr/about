Crowdminr uses a combination of statistical techniques. A global hazard function is calculated using survival analysis, specifically Kaplan-Meier estimation, giving the chance of a loan defaulting in each month through its term. Random forests are used to estimate a multiplicative frailty term, which you can think of as how generally broke the borrower is. From here you can calculate the probabilities of default in each month. Since money is worth less in the future than now, each payment is discounted by its Net Present Value Summing all the probabilities multiplied by their net present values gives an expected return for each loan.
